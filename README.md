# Greenden Website

Greenden is a simple and responsive plant store website built using **HTML** and **Tailwind CSS**. The project features a clean, modern design and demonstrates the use of utility-first CSS for creating responsive and visually appealing web pages.

## Features

* Responsive and modern user interface
* Utility-first styling with Tailwind CSS
* Home page showcasing featured plants and collections
* Product page displaying a variety of plants
* Contact page for customer inquiries
* Mobile-friendly layout

## Technologies Used

* HTML
* Tailwind CSS

## Pages

### Home

Displays collections, and promotional content.

### Products

Showcases a collection of plants with a clean and organized layout.

### Contact

Provides a contact section for users to get in touch.

## Purpose

This project was created to practice front-end web development using HTML and Tailwind CSS, with a focus on responsive design, reusable utility classes, and modern UI development.
